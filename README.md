# Sales Manager CRM – PWA

> React 18 + Vite PWA | Offline support | GitHub Pages hosted

## Quick Start
```bash
npm install
npm run dev       # dev server
npm run build     # production build
```

## Deploy
Push to `main` → GitHub Actions auto-deploys to GitHub Pages.  
Live URL: `https://akkibansode8563-boop.github.io/sales-crm/`
